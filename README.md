# Local Services Booking Platform

## 🔗 Live Demo
https://local-service-app-frontend.onrender.com

## 🚀 Features
- User authentication (JWT)
- Service booking system
- Booking history tracking
- Responsive UI

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## 📌 Description
This is a full-stack web application that allows users to book local services easily. It includes authentication, booking management, and a user-friendly interface.
